# VirtualMouseUsingHandGestures
A Python-based virtual mouse that uses hand gestures for control.

Built with: 

📌 MediaPipe – Hand tracking 

📌 OpenCV – Image processing

📌 PyAutoGUI – Mouse control 

Simple, efficient, and hands-free!
